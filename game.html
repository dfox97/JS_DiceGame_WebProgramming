Dice game

Choose number of dice to roll between 3 and 6

Rules of the game

Description Example (with four dice) Example Points Calculation
All dice have the same value (+60) 3,3,3,3 60 + 3 + 3 + 3 + 3 = 72
Dice have the same value apart from 1 (+40) 4,4,4,6 40 + 4 + 4 + 4 + 6 = 58
A run sequence(straight) (+20) 3,4,5,6 20 + 3 + 4 + 5 + 6 = 38
All dice have different values (not a run) (+0) 4,5,3,1 4 + 5 + 3 + 1 = 13
Other outcome 4,5,3,3 0
